# Tutorial Screenshots

Place your screenshots here with these exact filenames:

1. **main-interface.png** - Main interface showing Play button, knobs, and controls
2. **master-controls.png** - Master controls menu showing Play button, Length/Pause knobs, Volume knob, and QR Code button
3. **qr-code-display.png** - QR Code overlay showing the QR code for slave devices to scan
4. **slave-status-indicator.png** - Slave device status indicator showing connection status
5. **sound-preset-selector.png** - Sound Options dropdown showing available sound presets
6. **note-preset-selector.png** - Presets dropdown showing available note presets
7. **edit-mode-interface.png** - Edit mode showing editable note fields, remove buttons, and "insert new key" button
8. **sound-flakes-volumes.png** - Sound Flakes Volumes section showing individual volume sliders for each slave device

## How to Take Screenshots

1. Open the app in your browser (http://localhost:3000)
2. Navigate to the section you want to screenshot
3. Take a screenshot (Cmd+Shift+4 on Mac, or use browser dev tools)
4. Save it with the exact filename listed above
5. Place it in this directory

The tutorial will automatically use these images once they're added.
